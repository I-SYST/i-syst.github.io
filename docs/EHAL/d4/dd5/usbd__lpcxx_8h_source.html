71 // USBD setup request and endpoint event handler type.
72 //
73 // The application layer should define the custom class's EP0 handler with
74 // function signature. The stack calls all the registered class handlers on
75 // any EP0 event before going through default handling of the event. This
76 // gives the class handlers to implement class specific request handlers and
77 // also to override the default stack handling for a particular event targeted
78 // to the interface. If an event is not handled by the callback the function
79 // should return ERR_USBD_UNHANDLED. For all other return codes the stack
80 // assumes that callback has taken care of the event and hence will not process
81 // the event any further and issues a STALL condition on EP0 indicating error
82 // to the host.
83 // For endpoint interrupt handler the return value is ignored by the stack.
84 // Parameters:
85 // [in] hUsb Handle to the USB device stack.
86 // [in] data Pointer to the data which will be passed when callback function is called by the stack.
87 // [in] event Type of endpoint event. See USBD_EVENT_T for more details.
88 // Returns:
89 // The call back should returns ErrorCode_t type to indicate success or error condition.
90 // Return values:
91 // LPC_OK On success.
92 // ERR_USBD_UNHANDLED Event is not handled hence pass the event to next in line.
93 // ERR_USBD_xxx For other error conditions.
94 typedef uint32_t (*USB_EP_HANDLER_T)(USBD_HANDLE_T hUsb, void *data, uint32_t event);
