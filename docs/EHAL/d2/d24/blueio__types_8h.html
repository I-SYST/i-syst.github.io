Detailed Description

Contains definitions for blueio standard data types.

These data types are used in communication protocol to standardize data exchange format. For example :

  • The manufacturer specific data in BLE advertisement packet.
  • BLUEIO BLE Service data characteristic
  • Serial communications
  • and so on
