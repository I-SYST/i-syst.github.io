Detailed Description

Device base class.

This is the base class to implement all sort devices, hardware or software. For example a sensor device or a software audio decoder. The device can transfer data via it's DeviceIntrf object.

Set device's map address.

Device address is dependent of interface and device type. For I2C type it would be the 7 bits address, SPI would be CS pin index, other memory mapped would be a 32bit address.

Parameters
Addr: Device's address or zero based chip select index
