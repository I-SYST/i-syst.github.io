6 Desc : System status definitions & encoding.
7 
8  The system status keeps track of all system runtime states.
9 
10  it consists of a 32 bits value encoded as follow:
11 
12  31 28|27 16|15 8|7 0
13  +--------------+---------------------+-------------------+---------------+
14  | Type (4 bits)| Module ID (12 bits) | reserved (8 bits) | Code (8 bits) |
15  +--------------+---------------------+-------------------+---------------+
16 
17  All subsystems or library modules must reserve its module ID
18  in this file.
19 
20  The status value (encoded) of 0 is reserved for Ready/No error
21 
22  Type : 0000b Runtime states
23  0100b Warning (need attention)
24  1000b Non fatal error (recoverable)
25  1111b Fatal error (system halt)
26 
27  Each Type have a number number of common Codes for all modules
28  Those common codes are also defined and maintain here
29 
30  Module ID :
31 
32  ---- Applications
33  0x000 : Main application
34 
35  ---- Library modules
36  0x100 : OS runtime (Multi-Thread, IPC class, ObjectKernel)
37  0x102 : Component class
38  0x110 : File I/O class
39  0x111 : Multi-processor data transfer class
40  0x112 : Audio processor class
41  0x113 : Video processor class
42  0x114 : Mpeg processor class
43  0x115 : USB Device
44  0x116 : USB Host
